# Visual Design System: Pristine Laundry HQ

## 1. Public Portal Identity (Modern Luxury)
The public site is designed to feel "Pristine" — clean, spacious, and trustworthy.

- **Primary Color:** `#0F172A` (Slate 900) - Deep, authoritative navy.
- **Secondary Color:** `#F8FAFC` (Slate 50) - Crisp, clean white-ish background.
- **Accent Color:** `#3B82F6` (Blue 500) - Reliable, water-like blue.
- **Success/Eco:** `#10B981` (Emerald 500) - Sustainability and freshness.

**Typography (Google Fonts):**
- **Heading:** `Lexend` (Weight 700) - Modern, geometric, and ultra-legible.
- **Body:** `Inter` (Weight 400, 500) - Clean, neutral, high-readability.
- **Accent:** `Cormorant Garamond` (Italic) - For touch of traditional luxury.

**UI Style:**
- **Buttons:** 12px border-radius (Large), 1.5px border, subtle shadows.
- **Grid:** 12-column flex grid with 24px gutters.
- **Imagery:** High-resolution photography focusing on textures (silk, cotton) and smiling, professional staff.

## 3. UI Primitives Scale
- **Radius:** Small (4px), Base (8px), Large (12px), Full (9999px).
- **Shadows:** Soft (0 1px 2px), Elevated (0 10px 15px), Deep (0 20px 25px).
- **Transitions:** `duration-200 ease-in-out` for all interactive elements.
